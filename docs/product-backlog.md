# ProPath — product backlog (user stories)

**Team:** BlackCS · **Product:** ProPath (career preparation)

Prioritized list of MVP-aligned features. Each item is written as a user story.

| Priority | User story |
| -------- | ---------- |
| P1 | As a **new user**, I want to **create an account** so that **I can save my job search data securely**. |
| P1 | As a **user**, I want to **log in** so that **only I can see and edit my applications**. |
| P1 | As a **user**, I want to **add a job application** (company, role, status, next action) so that **I can track opportunities in one place**. |
| P2 | As a **user**, I want to **edit an application** so that **status and follow-up dates stay accurate**. |
| P2 | As a **user**, I want to **delete an application** so that **I can remove mistakes or closed leads**. |
| P2 | As a **user**, I want to **see my applications sorted by urgency** so that **I know what to do first**. |
| P2 | As a **user**, I want a **“today / this week” dashboard** so that **I can focus on imminent actions**. |
| P3 | As a **user**, I want to **connect Google Calendar** so that **interviews and fixed events appear in context**. |
| P3 | As a **user**, I want to **see upcoming calendar events next to my applications** so that **I can plan around my real schedule**. |
| P3 | As a **user**, I want **OAuth tokens stored securely on the server** so that **calendar access stays private**. |
